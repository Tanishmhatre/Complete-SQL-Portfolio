# Complete SQL Portfolio

**A comprehensive SQL portfolio showcasing real-world datasets and projects. Covers the full SQL spectrum: basic SELECT and filtering, DISTINCT, WHERE, ORDER BY, LIMIT, string & date functions, mathematical expressions, column aliases, aggregate functions (SUM, AVG, COUNT, MIN, MAX), GROUP BY, HAVING, JOINs (INNER, LEFT, RIGHT, FULL), subqueries, CTEs, window functions, and advanced SQL Server analytics. Ideal for demonstrating SQL proficiency and practical data analysis skills.**

---

## Overview
This repository demonstrates **practical SQL expertise** through structured projects on real-world datasets. It is designed to **showcase end-to-end SQL skills**, from basic querying to advanced analytics, making it perfect for learners and data professionals.

**Skills Highlighted:**
- SELECT, filtering, sorting, and DISTINCT  
- String and date functions  
- Mathematical expressions & column aliases  
- Aggregate functions: SUM, AVG, COUNT, MIN, MAX  
- GROUP BY & HAVING clauses  
- JOINs: INNER, LEFT, RIGHT, FULL  
- Subqueries & Common Table Expressions (CTEs)  
- Window & ranking functions  
- Advanced SQL Server analytics and real-world problem solving  

---

## Datasets
All projects use **real-world datasets** to simulate practical analytics scenarios:  
- `supermarket_sales.csv` — sales, branches, products, payments, and customer data  
- Additional datasets will be included progressively to expand the portfolio
